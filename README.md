# hamster-tg

一个 Telegram Bot，用于将图片、视频、GIF 和文件批量保存到本地。基于本地 Telegram Bot API 服务，支持大文件（最大 2GB）。

## 功能

- 保存图片、视频、GIF 和文件到本地目录
- 批量转发（相册/媒体组）只回复一条汇总消息
- 保存时显示"上传中"状态
- 通过 `/newfolder` 切换保存目录，按来源或主题分类整理
- 文件名冲突时自动追加 `_1`、`_2` 等后缀，不会覆盖已有文件

## Bot 命令

| 命令 | 说明 |
|------|------|
| `/start` | 显示帮助信息 |
| `/newfolder <名称>` | 创建并切换到指定文件夹 |
| `/status` | 查看当前文件夹和已保存文件数 |
| 直接发送媒体 | 自动保存到当前文件夹 |

文件夹名称支持字母、数字、中文、`-` 和 `_`。
